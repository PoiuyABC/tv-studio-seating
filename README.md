# 电视台演播大厅座次排序系统

TV Studio Seating Arrangement System

## 功能特性

- **可视化座次布局**：还原演播大厅座位分布
- **拖拽调换**：支持拖拽座位进行人员调换
- **信息编辑**：点击座位编辑人员详细信息（姓名、单位、职务、电话、备注）
- **Excel 导入/导出**：支持 Excel 文件批量导入导出
- **悬浮提示**：鼠标悬浮显示人员详细信息
- **响应式设计**：适配不同屏幕尺寸

## 座次布局

- **单排**：左9座 + 中20座 + 右9座 = 38座
- **双排**：左9座 + 中19座 + 右9座 = 37座
- **总计**：563座

### 座号编排

- **左侧**：37, 35, 33, 31, 29, 27, 25, 23, 21（单数从大到小）
- **中间**：19, 17, 15...1, 2, 4, 6...20/18（先单数后双数）
- **右侧**：22, 24...38 / 20, 22...36（双数从小到大）

## 使用方法

1. 点击空座位 → 填写人员信息 → 保存
2. 点击已占座位 → 编辑或删除人员
3. 拖拽已占座位到另一座位 → 交换位置
4. 鼠标悬浮 → 查看人员详情
5. 导出 Excel → 批量保存座次数据
6. 导入 Excel → 批量导入座次数据

## 技术栈

- HTML5 + CSS3 + JavaScript
- SheetJS (xlsx) - Excel 文件处理
- LocalStorage - 本地数据存储
